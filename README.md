# Listify
A way to check if your lists are valid to other lists

### Example usage:

| Database      | Search term   | Result |
| ---:          |     :---:     | :---   |
| Exceedingly   | Exceedingly   | Yes    |
| Symmetallism  | Symmetallism  | Yes    |
| Corynne       | Corynne       | Yes    |
| Purse-swollen | Purse-swollen | Yes    |
|               | Prepunches    | No     |
|               | Malory        | No     |
|               | Retelling     | No     |
|               | Pubescence    | No     |
